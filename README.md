# Hospital-Performance-Analytics-Project
A hospital data analytics project that uses SQL and Power BI to study patient records, billing, diagnoses, bed occupancy, and doctor performance to support better healthcare decisions

## Project Objective 
The objective of this project is to analyze hospital patient data using SQL and Power BI to understand patient admissions, diagnoses, billing, bed occupancy, doctor performance, and overall hospital operations. The project focuses on identifying patient trends, improving resource management, and evaluating hospital performance through interactive dashboards and data analysis.

By converting raw hospital data into meaningful insights, this project demonstrates the use of SQL for data analysis and Power BI for data visualization. The insights can help hospitals improve patient care, optimize bed utilization, manage hospital resources more efficiently, and support better decision-making through data-driven analysis.
